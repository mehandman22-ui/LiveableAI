# LiveableAI

Community-driven AI app for safe, private, and transparent AI interactions.

---

## Mission
LiveableAI empowers users to communicate and interact with AI safely, privately, and transparently. Our goal is to create a community-driven platform where technology serves everyone without compromising privacy or ethics.

---

## Features
- Cross-platform mobile app (iOS & Android) built with Flutter
- AI-powered chat interface for personal use
- Privacy-first approach; no sensitive user data stored unnecessarily
- Transparent community contributions and updates
- Optional unobtrusive ads or donation framework to cover operational costs

---

## Getting Started

### Prerequisites
- Flutter SDK installed ([Flutter installation guide](https://flutter.dev/docs/get-started/install))
- Git installed
- IDE (VSCode, Android Studio, or IntelliJ)

### Installation

```bash
git clone https://github.com/mehandman22-ui/LiveableAI.git
cd LiveableAI
flutter pub get
flutter run
```
